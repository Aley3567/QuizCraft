# 学习科学研究: 什么样的出题方式真正有效

> Exa Agent Run 4 - high effort - 2026-06-23

## 核心学习科学发现

### 1. 提取练习 (Retrieval Practice) — 效果最强
- **发现**: 测试不仅检测学习，还能提高记忆。主动回忆比被动重读效果显著更好
- **来源**: Dunlosky 2013, Agarwal 2021, Karpicke review, Nature Reviews Psychology 2022
- **对 QuizCraft 的意义**: 不能只出选择题（识别型），必须有简答/自由回忆（提取型）。频繁低风险测试 > 少数大考

### 2. 反馈是关键调节因素
- **发现**: 没有反馈的提取练习可能无效甚至有害（记住错误答案）
- **对 QuizCraft 的意义**: 每道题必须有验证过的答案 + 解释 + 来源引用。做不到的就不出

### 3. 精细化提问 (Elaborative Interrogation)
- **发现**: "为什么这是对的？""为什么 X 导致 Y？"类型问题效果中等偏高
- **对 QuizCraft 的意义**: 基础回忆题之后追加"为什么/怎么"类推理题

### 4. 交叉练习 (Interleaving) — 关键！
- **发现**: 混合不同知识点/题型比按章节顺序做题效果好得多，虽然感觉更难
- **对 QuizCraft 的意义**: **不要按文档顺序出题！** 混合新旧概念、相似概念、不同题型

### 5. 间隔练习 (Spacing/Distributed Practice)
- **发现**: 最可靠的学习科学发现之一。间隔 ≈ 目标记忆周期的 10-20%
- **对 QuizCraft 的意义**: 不是一次性出 100 张卡，而是分天推送。让用户设考试日期，自动调度

### 6. 连续再学习 (Successive Relearning)
- **发现**: 间隔 + 提取 组合效果最强：回忆到掌握 -> 反馈 -> 间隔后再回忆
- **对 QuizCraft 的意义**: 最优循环 = 小块学习 -> 提取 -> 反馈 -> 修复 -> 间隔再学 -> 混合复习

### 7. 学生偏爱低效方法
- **发现**: 划重点/重读感觉有在学但实际效果差
- **对 QuizCraft 的意义**: 默认模式就是主动回忆，摘要只是辅助

### 8. AI 出题可能削弱学习效果
- **发现**: 自己出题/整理的过程本身有学习价值，纯机器生成可能去除这个好处
- **对 QuizCraft 的意义**: 让用户审核/编辑/打标签——保留认知参与

## 出题质量标准

### 什么是好题
- 来源锚定、围绕学习目标、可回答、清晰、适当难度、配反馈
- 混合格式：自由回忆(最强记忆效果) + 简答(解释) + MCQ(辨别) + 应用场景(迁移)
- 每题包含：预期答案、解释、来源引用、Bloom 层级、难度估计、错误修复反馈

### 常见低质量模式
- 表面事实题（只考定义背诵）
- 从标题抄的问题而非核心内容
- 不支持/幻觉答案
- 模糊多解题目
- 措辞泄露答案
- 干扰项明显错误或有规律
- 只有文档顺序出题没有交叉
- 没有反馈/解释

### Bloom 层级指导
| 层级 | 出什么题 |
|------|---------|
| 记忆 | 回忆定义/事实/公式 |
| 理解 | 用自己的话解释/比较/总结 |
| 应用 | 在新但受限场景中使用概念 |
| 分析 | 区分组件/假设/因果/诊断原理 |
| 评价 | 用课程标准判断一个说法/方法 |
| 创造 | 设计方案/模型/实验/论证 |

## Prompt 工程最佳实践

1. **来源锚定 + 证据跨度**: 要求每题引用源文段落，不支持的就不出
2. **学习目标先行**: 先提取 8 个最核心概念，再每概念出 2 题（1 回忆 + 1 应用）
3. **Bloom 控制**: 每层各出一题，要求模型解释为什么是这个层级
4. **Few-shot 示例**: 给 2-5 个高质量样题，尤其是高阶分析/评价题
5. **自我批评管线**: 生成 12 题 -> 按 6 维度自评 1-5 分 -> 淘汰 <4 分的
6. **误解干扰项**: MCQ 干扰项来自常见误解而非随机，每个干扰项附解释

## 最优学习循环设计

```
1. 导入文档 -> 按学习目标分段
2. 每段生成小题集：回忆 + 为什么/怎么 + 应用 + MCQ + 交叉题
3. 学生先答题再看反馈
4. 反馈：纠错 + 源文引用 + 解释 + 缺口诊断
5. 错题：只重读相关段落 + 写自我解释
6. 进入间隔复习：错/低信心早回，对/高信心晚回
7. 每次复习混合新旧 + 交叉
8. 考前：模拟考试匹配预期题型和 Bloom 深度
```

## 反模式（绝对要避免）

- 上传 PDF 一次性生成巨大卡片组
- 给摘要不强制回忆
- 只出定义/事实题
- 按文档顺序出题
- 识别就标记完成（没有提取验证）
- AI 输出不验证来源
- 隐藏答案但不给反馈
- 过度依赖 MCQ + 弱干扰项
- 按数量而非遗忘/掌握来调度

## 真实用户投诉 (跨平台收集)

1. AI 生成的学习材料有时是"巨大的时间浪费"——还得检查不相关/无来源的题
2. AI 题目太通用/太简单/不像老师的真题风格
3. AI 闪卡太浅，把复杂概念简化成一行
4. AI 幻觉、误读上传内容、把对的答案判错
5. 付费墙、免费版限制、订阅摩擦
6. Bug：学习集丢失、同步慢、大卡组卡顿
7. 离线限制影响通勤/弱网复习
8. OCR/图片/手写/公式解析差
